### 📦 Getting Started  
1. Clone the repo:  
   ```bash
   git clone https://github.com/Hello-ship-code/react-apps.git
   ```  
2. Navigate into the project folder:  
   ```bash
   cd react-apps
   ```  
3. Install dependencies:  
   ```bash
   npm install
   ```  
4. Run the project:  
   ```bash
   npm run dev
   ```
